# Маркетплейс на Django и Flutter

Этот проект представляет собой полнофункциональный маркетплейс, состоящий из бэкенда на Django и мобильного приложения на Flutter.

## Бэкенд (Django)

Бэкенд построен на Django с использованием Django REST Framework для создания REST API.

**Основной функционал:**
- Регистрация и аутентификация пользователей (включая JWT)
- Просмотр категорий и товаров
- Добавление товаров в корзину и управление ею
- Оформление заказов
- Профиль пользователя

**Как запустить (требуется Docker):**

1.  Убедитесь, что Docker запущен.
2.  Выполните команду в корневой папке проекта:
    ```bash
    docker-compose up --build backend
    ```
3.  API бэкенда будет доступно по адресу `http://localhost:8000`.

## Фронтенд (Flutter)

Мобильное приложение на Flutter для взаимодействия с API бэкенда.

**Как запустить (требуется Flutter SDK):**

1.  Перейдите в папку с приложением:
    ```bash
    cd ecommerce_app
    ```
2.  Запустите приложение на эмуляторе или физическом устройстве:
    ```bash
    flutter run
    ```
